# MIPS64 gdbserver Build

This repository builds a static MIPS64 gdbserver binary using GitHub Actions.

## Usage

1. Push to main branch or manually trigger workflow
2. Check Actions tab for build status
3. Download the compiled `gdbserver-mips64` binary from artifacts

## Binary Details

- Architecture: MIPS64 (little-endian)
- Compilation: Static (no external dependencies)
- GDB Version: 14.1
